# Pan-Card-Ocr

This project extracts information from Indian PAN cards using OCR.

## Features

- PAN Number detection
- Name extraction
- Father's name extraction
- DOB detection

## Installation

1. Install Tesseract

Windows:
https://github.com/tesseract-ocr/tesseract

2. Install dependencies

pip install -r requirements.txt

## Run

python app.py

## Output

Extracted data will be saved in:

output/extracted_data.json
